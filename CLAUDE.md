# CLAUDE.md

이 저장소의 공통 규칙(커밋/PR 제목·본문, PR 리뷰)은 @AGENTS.md 를 단일 소스로 따른다.
규칙을 여기에 중복하지 않는다. 규칙 변경은 항상 `AGENTS.md`에서 한다.

## Claude Code 전용 메모

- 응답, PR 리뷰 코멘트, 커밋/PR 설명은 한국어로 작성한다.
  코드, API 이름, 파일명, 명령어, 에러 메시지는 영어 원문을 유지한다.
- 루트는 인덱스 역할만 한다. Python 실행 환경/의존성/`uv.lock`은
  각 학습 디렉터리(`레디스/`, `코딩테스트/`) 안에서 독립적으로 관리한다.
  명령은 해당 디렉터리 안에서 `uv run ...` 으로 실행한다.
- 하위 디렉터리에 `AGENTS.md` 또는 `CLAUDE.md`가 있으면 루트 규칙을 먼저
  적용한 뒤 하위 규칙을 추가로 반영한다.
