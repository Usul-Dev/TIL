# Redis 기초

Redis는 인메모리 키-값 저장소다. 데이터를 디스크가 아니라 메모리(RAM)에 두기 때문에
읽기/쓰기가 매우 빠르다. 캐시, 세션 저장소, 순위표, 메시지 큐 등에 쓴다.

## 자료구조

Redis는 단순 문자열뿐 아니라 List, Set, Sorted Set(ZSet), Hash 등 여러 자료구조를 지원한다.
예를 들어 Sorted Set은 점수 기반 정렬을 제공해 실시간 순위표 구현에 적합하다.

## 영속성

인메모리지만 데이터를 잃지 않도록 두 가지 영속성 옵션이 있다.
- RDB: 특정 시점의 스냅샷을 파일로 저장한다.
- AOF(Append Only File): 모든 쓰기 명령을 로그로 기록해 재시작 시 재생한다.
