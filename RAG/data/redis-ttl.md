# Redis TTL과 만료

TTL(Time To Live)은 키의 만료 시간이다. `EXPIRE key seconds`로 키에 수명을 부여하면,
지정한 시간이 지난 뒤 Redis가 그 키를 자동으로 삭제한다. 세션·캐시처럼 일정 시간만
유효해야 하는 데이터에 쓴다.

## 만료 동작 방식

Redis는 만료된 키를 두 가지 방식으로 지운다.
- Lazy(수동): 키에 접근할 때 만료됐으면 그때 삭제한다.
- Active(능동): 백그라운드에서 주기적으로 만료된 키를 일부 표본으로 검사해 삭제한다.

## 주의점

TTL을 설정한 키를 다른 값으로 덮어쓰면(SET 등) 기존 TTL이 사라질 수 있다.
값만 갱신하고 만료는 유지하려면 `KEEPTTL` 옵션을 함께 쓴다.
